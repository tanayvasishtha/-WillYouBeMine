# WillYouBeMine

A small Valentine's Day page: a short story, then one question. Play/pause music from the header; use the Spotify strip in the corner for next track.

## Run locally

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Stack

Next.js 16, React 19, Tailwind CSS. Music via Spotify embed (no backend).
